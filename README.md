# AnyToolLab

## To Run This Project Locally
```bash
git clone https://github.com/MKisil/anytoollab.git
```
```bash
cd anytoollab
```
```bash
docker compose up
```
After setup, visit [127.0.0.1](http://127.0.0.1) in your browser.

## About This Project
This is a website that is a collection of various web tools.

## Tools
The following tools are currently available on the site[^1]:
- PDF Tools
  - PDF editor
  - PDF split
  - PDF add page numbers
  - PDF rotate
  - PDF delete pages
  - PDF encrypt
  - PDF decrypt
  - PDF extract text
  - Convert to PDF
    - PNG to PDF
    - JPG to PDF
